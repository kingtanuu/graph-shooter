# グラフシューター 〜関数で、撃ち抜け。〜

座標平面に浮かぶ星を、自分で考えた **y = f(x)** のグラフで撃ち抜くパズルシューティング。
小6の比例から高2の三角関数まで、「式とグラフの対応」を体で覚える学習ゲームです。

第7回 学力向上アプリコンテスト応募用プロトタイプ。応募プラン全体は [../../PLAN.md](../../PLAN.md) を参照。

## 動作方法

ビルド不要の純粋な HTML/CSS/JavaScript です。

```bash
# リポジトリのこのフォルダで
python3 -m http.server 5173
# → http://localhost:5173 を開く
```

（`index.html` をブラウザで直接開いても動作します）

## 遊び方

1. ステージを選ぶ
2. 星の座標を見て、全部を通るグラフの式を考える
3. `y =` に式を入力（例: `0.5x + 1`、`0.25x^2`、`abs(x-1)+2`、`2sin(pi*x/2)`）
4. **発射🚀**（Enterキーでも発射）— グラフが弾道になって左から描かれる
5. 星に触れると獲得、赤いブロックに当たると爆発。全部の星を集めればクリア
6. PAR以内の発射回数でランクS！

- 序盤のステージは入力中に弾道の**リアルタイムプレビュー**が出ます（式を変えるとグラフがどう変わるか体感できる）
- 終盤は「ノーレーダー戦」＝プレビューなし。頭の中でグラフを描こう
- 💡ヒント と 答えの例、クリア後の「📖学びメモ」で数学的ポイントを解説

## 使える記法

| 書き方 | 意味 |
|---|---|
| `2x`, `0.5x+3` | 暗黙の乗算OK |
| `x^2` | べき乗 |
| `sin( ) cos( ) tan( )` | 三角関数 |
| `abs( )` | 絶対値 |
| `sqrt( )` | 平方根 |
| `log( ) exp( )` | 自然対数・指数 |
| `pi` または `π`, `e` | 円周率・自然対数の底 |

## テスト

数式パーサとステージ整合性（模範解答が星を通り障害物に当たらないこと）のユニットテスト:

```bash
node --test tests/parser.test.mjs
```

## 技術メモ

- 数式パーサは自作（トークナイザ + 操車場アルゴリズム）。`eval`/`Function` 不使用で安全
- Canvas 2D 描画、Web Audio 効果音（音源ファイル不要）、`localStorage` に進捗保存
- `prefers-reduced-motion` 対応、スマホ対応（入力補助チップつき）
- BGMはタイトル・ステージ選択画面のみで再生し、プレイ中は自動停止（🔊ボタンでミュート可）
